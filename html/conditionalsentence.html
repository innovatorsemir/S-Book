Conditional Sentence

⇒ A Conditional sentences is a sentence with two clauses. The condition clause which begins with If clause and the main or result clause.

⇒ There are four main types of conditional sentence in English:

A. Zero Conditional B. First Conditional C. Second Condtional D. Third Conditional

Additional C.Sentences

E. Mixed 1 Conditional F. Mixed 2 Conditional

⇒ We use these forms:

  • to talk something based on condition:

E.g: If babies are hungry, they cry
→ If I study today, I go the party.
→ I will be sad if my football team loses the match.
→ If she fell, she would hurt herself.
→ If she had studied, she would have passed the exam. (but, really we know she didn't study and so she didn't pass)



A. Zero Conditional

⇒ We use this when results of things is always true. Like a scientific fact / general truth.

⇒ Form:-
If + Present Simple, Present Simple

⇒ OR
If + Past Simple, Past Simple

If clause Main clause
Present simple
(S + V1/Vs + O)		 Present simple
(S + V1/Vs + O)
E.g:1) If we don't eat food, we die.
E.g:2) If you heat water at 100 degrees, it boils.
E.g:1) If I have time, I study English.
E.g:3) Years ago if people travelled a long distance, the journey took days.


B. First Conditional

⇒ To talk about possibilities and also very likely in the present or in the future.

⇒ Form:-
If + Present Simple, Future Simple

⇒ OR
If + Present Perfect, Future Simple

If clause Main clause
Present simple
(S + V1/Vs + O)		 Future simple
(S + will/shall + V1 + O)
E.g:1) If I find her address, I will send her an invitation.
E.g:2) If you see her, don't tell her I was here.
E.g:3) If it is hot today, we may go swimming.
E.g:4) If you have studied for this exam, you will pass it.

Note: "If" may be replaced by "When". It used we sure on that.

Compare these two sentences

E.g:1) If I go out, I will call you.

C. Second Conditional

⇒ There is an unreal possibility or imaginary situations in the present or future.

⇒ Form:-
If + Past Simple, Present Conditional

If clause Main clause
Past simple
(S + V2 + O)		 Present conditional
(S + would/should + V1 + O)

Note: Present Conditional: would / wouldn't + V(bare form). For more click
Past/Perfect Conditional: would / wouldn't + have + past participle. For more click

E.g:1) If I were you, I wouldn't buy that expensive laptop.
E.g:2) If I won a million dollars, I would buy a new car.
E.g:3) If they left early, they might get to the theatre on time.
E.g:2) If I went to the supermarket, I could get some bread.


D. Third Conditional

⇒ The important thing about the third conditional is that both the condition and result are impossible now. Imagine situations in the past.

⇒ Form:-
If + Past Perfect, Past Conditional

If clause Main clause
Past perfect
(S + had + V3 + O)		 Past conditional
(S + would/should + have+ V3 + O)
E.g:1) If the weather had been good, we would have gone water-skiing.
E.g:2) We would have come yesterday if we had known about it.
E.g:3) If I had gone to the supermarket, I could have bought some bread.
E.g:4) If I had waited longer last night, I might have found out about it.


E. Mixed 1 Conditional

⇒ Present result of a past condition.

⇒ Form:-
If + Past Perfect, Present Conditional

If clause Main clause
Past perfect
(S + had + V3 + O)		 Present conditional
(S + would/should + V1 + O)
E.g:1) If we don't eat food,we will die.
E.g:2) If I had listened to your advice, I wouldn't be in the mess.


F. Mixed 2 Conditional

⇒ Past result of a present or continuing condition.

⇒ Form:-
If + Past Simple, Past Conditional

If clause Main clause
Past simple
(S + V2 + O)		 Past conditional
(S + would/should + Have+ V3 + O)

E.g:1) If we don't eat food,we will die.
E.g:2) If I were a good cook, I would have invited them to dinner.

Other Conditionals

⇒ Unless (if...not), in case, provided / providing (that), as/so long as.

Examples:
→ I will go swimming tomorrow unless it rains.
→ I'll take my umbrella with me in case it rains.
→ You can play with you friends in your room provided that you tidy up afterwards.
→ I'll help you as long as you come with me to the cinema.
